# 组合模式

## 意图

> 组合模式统一对象和对象集，使得使用相同接口使用对象和对象集。

## 关键代码

> 对象间有相同的接口

## 应用实例

> 组合模式常用于树状结构，用于统一叶子节点和树节点的访问，并且可以用于应用某一操作到所有子节点
