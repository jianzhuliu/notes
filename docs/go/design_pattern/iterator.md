# 送代器模式

## 意图

> 送代器模式用于使用相同方式送代不同类型集合或者隐藏集合类型的具体实现。

## 应用场景

> 可以使用送代器模式使遍历同时应用送代策略，如请求新对象、过滤、处理对象等
